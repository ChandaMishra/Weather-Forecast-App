  **Overview**
**Weather Forecast App** is a dynamic, user-friendly web application that provides real-time weather updates and a 7-day forecast for cities worldwide. Built with HTML, CSS, and JavaScript, it fetches data from the OpenWeatherMap API to display accurate weather conditions, including temperature, humidity, wind speed, and atmospheric pressure. The app features intuitive icons for weather visualization, a responsive design for seamless use across devices, and robust error handling for invalid searches.
